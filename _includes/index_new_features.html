Play 2.6.0-RC2 and Bootstrap 4 (alpha) support

The project has been split to give support for Play 2.6.0-RC2 and Bootstrap 4 alpha. So now there are 6 parallel versions depending on which version you need for Play Framework and Bootstrap:

  • 1.1.2-P26-B3-SNAPSHOT: Built for Play 2.6.0-RC2 and Bootstrap 3.3.7.
  • 1.1.2-P26-B4-SNAPSHOT: Built for Play 2.6.0-RC2 and Bootstrap 4.0.0-alpha.6.
  • 1.1.2-P25-B3-SNAPSHOT: Built for Play 2.5.15 and Bootstrap 3.3.7.
  • 1.1.2-P25-B4-SNAPSHOT: Built for Play 2.5.15 and Bootstrap 4.0.0-alpha.6.
  • 1.1.2-P24-B3-SNAPSHOT: Built for Play 2.4.11 and Bootstrap 3.3.7.
  • 1.1.2-P24-B4-SNAPSHOT: Built for Play 2.4.11 and Bootstrap 4.0.0-alpha.6.

Note! There is a selector on the topbar's right side for change between all versions.
