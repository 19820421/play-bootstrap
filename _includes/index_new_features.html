New features

Play 2.5.6 and Bootstrap 4 (alpha) support

The project has been split to give support for Bootstrap 4 alpha. So now there are four parallel versions depending on which version you need for Play Framework and Bootstrap:

  • 1.1-P25-B3: Built for Play 2.5.6 and Bootstrap 3.3.7.
  • 1.1.1-P25-B4-SNAPSHOT: Built for Play 2.5.6 and Bootstrap 4.0.0-alpha.5.
  • 1.1-P24-B3: Built for Play 2.4.8 and Bootstrap 3.3.7.
  • 1.1.1-P24-B4-SNAPSHOT: Built for Play 2.4.8 and Bootstrap 4.0.0-alpha.5.

Note! There is a new selector on the topbar's right side for change between all versions.

HTML5 data form validation

Field constraints are now automatically added as HTML5 data form validation.

CSRF support

Added CSRF support adding token to forms with formCSRF helpers. See more information here.

Internationalization improvements

Arguments are now automatically internationalized ('_label, '_hiddenLabel, 'placeholder, '_help, '_success, '_warning, '_error, '_text for checkbox, texts for radio, radioOption, select and selectOption).
