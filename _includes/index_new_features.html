New support for Bootstrap 4 (alpha)

The project has been split to give support for Bootstrap 4.0.0-alpha. So now there are three parallel versions depending on which version you need for Play Framework and Bootstrap:

  • 1.0-P24-B4-SNAPSHOT: Built for Play 2.4.4 and Bootstrap 4.0.0-alpha.
  • 1.0-P24-B3-SNAPSHOT: Built for Play 2.4.4 and Bootstrap 3.3.5.
  • 1.0-P23-B3-SNAPSHOT: Built for Play 2.3.10 and Bootstrap 3.3.5.

Note! There is a new selector on the topbar's right side for change between all versions.
