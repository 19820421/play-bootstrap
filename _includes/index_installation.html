Installation

You only need to add it as a dependency in your build.sbt file. 

{% if version.snapshot %}// Resolver is needed only for SNAPSHOT versions
resolvers += "Sonatype OSS Snapshots" at "https://oss.sonatype.org/content/repositories/snapshots/"

{% endif %}libraryDependencies ++= Seq(
  "com.adrianhurt" %% "{{ version.artifact }}" % "{{ version.code }}{% if version.snapshot %}-SNAPSHOT{% endif %}"
)

Note the play-bootstrap has filters, Bootstrap and jQuery as dependencies, so they are included automatically. Specifically, it adds the following dependencies for you: 

filters,
"org.webjars" % "bootstrap" % "{{ version.bootstrap_dependency }}",
"org.webjars" % "jquery" % "{{ version.jquery_dependency }}"

You can use different versions of jQuery or even Bootstrap. For example, if you prefer to use jQuery 2.1.0: 

"com.adrianhurt" %% "play-bootstrap" % "{{ version.code }}{% if version.snapshot %}-SNAPSHOT{% endif %}" exclude("org.webjars", "jquery"),
"org.webjars" % "jquery" % "2.1.0"
